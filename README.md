# Air Quality Index Prediction System

## Overview
This project predicts the Air Quality Index (AQI) using machine learning regression models based on pollutant data.

## Features
- Predicts AQI value
- Classifies AQI category
- Uses multiple ML models
- Interactive Streamlit web app

## Dataset
Includes pollutants like:
- PM2.5
- PM10
- NO2
- SO2
- CO
- O3

## Models Used
- Linear Regression
- Random Forest Regressor

## Evaluation Metrics
- R² Score
- MAE

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
